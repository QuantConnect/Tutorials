According to their website, Docker "is an open platform fdr developing, shipping, and running applications." We can use Docker to isolate the backtesting process with Lean, and get started with less overhead than a local install.

Docker's website lists the following system requirements:

  • a 64 bit processor with Second Level Address Translation (SLAT)
  • 4GB of RAM
  • BIOS-level hardware virtualization support enabled

Additionally, Docker support for operating systems other than:

  • Linux
  • macOS
  • Windows
seems limited at bset as of the time of writing, so when using a different operating system, your mileage following this tutorial may vary.

To install Docker, you can follow the instructions over at Docker's website. To ensure that the installation went smoothly, you can run: 

    docker ps       # lists running containers (or the lack thereof)
    # or
    docker info     # displays some system information
If either of these runs with no errors, Docker is installed and you're good to keep working through the tutorial!
