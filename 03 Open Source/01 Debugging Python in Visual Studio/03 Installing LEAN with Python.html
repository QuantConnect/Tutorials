The first step is to install LEAN and get it running with python algorithms.

To do this install:

  • Download Lean at the open-source project page.
  • Install Visual Studio.
  • Open QuantConnect.Lean.sln in Visual Studio
  • Build the solution by clicking Build Menu -> Build Solution
  • Press F5 to run

By default, Lean will run the C# version of BasicTemplateFrameworkAlgorithm.

Next step is to run the python version of BasicTemplateAlgorithm successfully.

To run that algorithm:

  • Edit config.json with the following paramenters:
    •       "algorithm-type-name": "BasicTemplateAlgorithm",
      "algorithm-language": "Python",
      "algorithm-location": "../../../Algorithm.Python/BasicTemplateAlgorithm.py",
  • Build the solution by clicking Build Menu -> Build Solution
  • Press F5 to run
